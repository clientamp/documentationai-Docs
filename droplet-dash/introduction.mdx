Droplet Dash is a high-performance, native macOS application designed to be the "**Cursor for DevOps**." It provides a powerful, SSH-based interface for managing your VPS servers, combining traditional command-line control with modern AI assistance, monitoring, and automated deployment pipelines.

## Modern Server Management

Droplet Dash moves beyond simple terminal emulators by providing a structured, intelligence-driven workflow for server administration.

- **Process-based SSH**: Uses the native `/usr/bin/ssh` with **ControlMaster** connection reuse for lightning-fast command execution and native support for **Tailscale SSH**.
- **On-Device Data**: All server metadata and connection configurations are stored locally on your Mac using **SwiftData**.
- **AI-Powered DevOps**: Integrated AI assistance helps you diagnose server issues, suggest security hardening steps, and generate deployment YAMLs.

## Security & Privacy

Built with a security-first mindset:

- **Keychain Integration**: All sensitive API tokens and credentials are stored securely in the **macOS Keychain**.
- **Shell Sanitization**: Every command sent to your servers is automatically sanitized to prevent shell injection attacks.
- **Hardened Runtime**: Droplet Dash utilizes the macOS Hardened Runtime to ensure the integrity of its local processing environment.

## Tech Stack

Droplet Dash is built using the latest Apple frameworks and a shared core architecture:

- **UI**: 100% SwiftUI (macOS 14+)
- **Data**: SwiftData for local-first persistence
- **Concurrency**: Swift 6 structured concurrency (Actors, TaskGroups)
- **Shared Architecture**: Leverages **SharedCore** for AI, Networking, and Security primitives.
