**Mac-First Course Publisher** is a native macOS application designed for course creators who prioritize a streamlined, local-first authoring experience. By combining a powerful macOS folder bundle structure with a local AI "co-pilot," it allows for deterministic, machine-readable course creation and high-quality educational delivery.

## Local-First Authoring

The platform is built on the `.coursepub` bundle format, ensuring your content remains accessible, versionable, and secure:

- **Folder Bundles**: Courses are stored as standard macOS folders containing JSON metadata and Markdown content.
- **Deterministic AI**: A local AI API contract allows agents to assist in course creation, lesson drafting, and quiz generation.
- **Static-First Delivery**: Courses are optimized for lean, static-first delivery to learners, avoiding bloated LMS backends.

## Technical Foundation

- **Platform**: Native macOS (Swift 6.0+, SwiftUI)
- **Format**: `.coursepub` bundle (JSON + Markdown + Assets)
- **AI**: Local LLM integration via AIProvider
- **Review**: Built-in integrity checks for broken links and missing assets
