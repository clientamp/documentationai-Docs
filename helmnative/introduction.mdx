HelmNative is a high-performance email client built exclusively for macOS using **SwiftUI** and **Swift 6**. It combines professional email management with a powerful "cloudless" AI experience, connecting directly to your IMAP servers without intermediate cloud storage or proprietary backends.

## Privacy First

Unlike many modern email clients, HelmNative respects your data privacy. All emails, server credentials, and database records stay local on your Mac.

- **Direct IMAP Sync**: High-speed synchronization using native macOS `Network.framework` connections.
- **Keychain Security**: All passwords and API keys are stored securely in the **macOS Keychain**.
- **Local Persistence**: All data is stored in a performant local SQLite database via **GRDB.swift**.

## Integrated Task Management

HelmNative includes a built-in **Todo List** that is deeply integrated with your email workflow. Transform any email into a task with a single click, maintaining a permanent link back to the original message.

## Tech Stack

Built for the future of macOS development:

- **UI**: 100% SwiftUI (macOS 14+)
- **Logic**: Swift 6 with strict concurrency
- **Persistence**: SQLite + GRDB.swift (ValueObservation & FTS5)
- **Networking**: native `Network.framework` (NWConnection)
- **Package Architecture**: **HelmCore** (Shared logic) + **HelmMac** (App target)
