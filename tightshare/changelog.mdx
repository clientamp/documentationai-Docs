<Update label="2026-03-15" description="v1.0.0 Release" tags={["feature"]}>
  ## Initial Release
  
  TightShare (PostFlow) is now live as a native SwiftUI automation tool! Our first version includes:
  - YouTube video to blog post conversion.
  - Direct integration with **Ghost CMS**.
  - AI-powered content refinement via **Anthropic API**.
  - **LickityClick** integration for tracked affiliate links.
  - Local-first data management with **SwiftData**.
  - Secure credential storage in the macOS Keychain.
</Update>
