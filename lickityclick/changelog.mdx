<Update label="2026-03-16" description="Campaign Metrics & Hardening" tags={["feature", "improvement", "testing"]}>
  ## What's New
  
  ### Campaign Metrics Integration
  - **Dashboard Wiring**: Successfully wired the campaign metrics API into the dashboard UI. You can now see aggregate clicks, conversions, and performance data directly on campaign detail pages.
  - **Visual Polish**: Tightened up the visual hierarchy, formatting, and edge-case display for all metrics cards to ensure a trustworthy and professional look.

  ### UI/UX Improvements
  - **Loading Skeletons**: Added deliberate loading skeletons for all campaign metrics surfaces to reduce layout shift and provide a more responsive feel.
  - **Empty States**: Implemented intentional empty-state and zero-data UX for analytics views, ensuring the dashboard looks polished even before data starts flowing.
  - **Mobile Resilience**: Audited and fixed wrapping, spacing, and hierarchy issues for metrics views on small and narrow screens.

  ### Reliability & Testing
  - **Integration Tests**: Added a comprehensive integration test suite covering campaign CRUD operations and metrics retrieval.
  - **Smoke Coverage**: Implemented lean smoke tests for campaign metrics data flow to prevent regressions in these high-value surfaces.
  - **UI Audit**: Conducted a full design token compliance audit to ensure consistency across spacing, typography, and color usage.
</Update>

<Update label="2026-03-15" description="v1.0.0 Release" tags={["feature"]}>
  ## Initial Release
  
  LickityClick is now live! Our first version includes:
  - Full Shlink v4 integration for link routing and management.
  - Intelligent UTM builder with real-time preview.
  - Campaign grouping and basic tag management.
  - Real-time analytics for Geo, Device, Browser, and Referrers.
  - Downloadable QR codes.
  - Smart Redirect Rules (Device, Language, Query Params).
  - Multi-domain support.
  - Shareable Public Reports for sponsors.
</Update>
