<Update label="2026-03-15" description="v1.0.0 Release" tags={["feature"]}>
  ## Initial Release
  
  Droplet Dash is now live as a native SwiftUI DevOps companion! Our first version includes:
  - Process-based SSH management with **ControlMaster** connection reuse.
  - Native support for **Tailscale SSH**.
  - High-performance **SSH Connection Pool** for multi-server sessions.
  - **System Monitoring** for CPU, Memory, Disk, and Docker containers.
  - YAML-based **App Catalog** and automated deployment engine.
  - Built-in **Shell Sanitization** to prevent command injection.
  - Secure credential storage in the macOS Keychain via **SharedCore**.
  - Local-first persistence using **SwiftData**.
</Update>
