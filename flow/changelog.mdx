<Update label="2026-03-15" description="v1.0.0 Release" tags={["feature"]}>
  ## Initial Release
  
  Flow is now live as a Things 3 inspired task management platform! Our first version includes:
  - High-level **Areas** and actionable **Projects**.
  - **Headings** and **Tasks** with integrated checklists.
  - **Natural Language** date parsing for rapid scheduling.
  - Real-time collaboration powered by **Supabase**.
  - **Rich Text** task notes using Tiptap.
  - Fluid **Drag & Drop** reordering with fractional indexing.
  - Fully responsive design with dark mode support.
</Update>
