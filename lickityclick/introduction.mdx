LickityClick is a professional link management dashboard designed for creators and marketers. Built as a high-performance Next.js 16 application on top of the [Shlink](https://shlink.io) link engine, it provides advanced UTM building, campaign grouping, real-time analytics, and shareable sponsor reports.

## Core Value

While many URL shorteners focus on simple redirects, LickityClick is built to help you **prove ROI**. By grouping links into campaigns and providing deep analytics across countries, devices, and referrers, you can give your sponsors the data they need to keep renewing deals.

## Tech Stack

LickityClick uses a modern, self-hosted stack designed for speed and reliability:

- **Framework**: Next.js 16 (App Router, Server Actions, React Server Components)
- **Database**: PostgreSQL 17 + Redis 7
- **ORM**: Drizzle ORM
- **Auth**: Better Auth (Email/Password)
- **Charts**: Apache ECharts
- **Link Engine**: Shlink v4 (Dockerized REST API v3)
