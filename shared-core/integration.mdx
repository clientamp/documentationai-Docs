## Agent-First API Posture

All internal APIs must adhere to the following rules to ensure they are easily consumable by both human developers and AI agents:

1. **REST+JSON Only**: Deterministic, easy-to-parse payloads.
2. **Envelope Pattern**: All responses wrapped in a consistent metadata envelope.
3. **Idempotent Writes**: All state-changing operations must support idempotency keys.
4. **Cursor-Based Pagination**: For stable retrieval of large datasets.
5. **Typed Enums**: Avoid generic strings for states or categories.
6. **ISO 8601 Dates**: Standardized timestamp format.
7. **Revenue in Cents**: Avoid floating-point errors by using integer cents.
